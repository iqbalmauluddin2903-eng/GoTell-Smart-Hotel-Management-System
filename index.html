Aplikasi manajemen hotel berbasis console/CLI yang dibuat menggunakan C++ murni. Mensimulasikan operasional hotel nyata dengan tiga role utama.

Language · C++ Type · Console App Login · Role Based Platform · Windows Status · Development

// fitur_utama

Apa yang bisa GoTell lakukan?

👔
Manager
Dashboard pendapatan real-time, statistik okupansi per tipe kamar, laporan transaksi, dan monitoring status seluruh kamar hotel.
🧾
Resepsionis
Check-in & check-out tamu, tambah layanan ke transaksi aktif, cetak struk, dan simulasi pembayaran Tunai / QRIS.
🧹
Housekeeping
Kelola status kamar — selesaikan pembersihan, set maintenance, dan tandai kamar kembali siap pakai.
🔐
Validasi Input
Nama hanya huruf & spasi, KTP 16 digit angka, HP diawali 08/62, parsing angka manual tanpa stoi(), maks. 3x login.
⌨️
Navigasi Interaktif
Menu kotak dengan navigasi tombol panah atas/bawah + Enter. Animasi loading dan warna ANSI di seluruh tampilan.
💳
Struk & QRIS
Cetak struk transaksi lengkap dengan pajak 11% dan service charge 5%. Simulasi tampilan QRIS berbasis simbol ASCII.
